# Getting Started Guide

This guide will help you understand how to use this markdown publishing system.

## Adding New Pages

To add a new page, simply create a `.md` file in the `content/` directory.

## Writing Content

You can use standard markdown syntax:

### Lists

- Item 1
- Item 2
- Item 3

### Code Blocks

```javascript
console.log('Hello, world!');
```

### Links

You can add [links](https://example.com) to external resources.

## Frontmatter

Add metadata to your pages using frontmatter:

```yaml
---
title: My Custom Title
---
```

If you don't provide a title, the filename will be used.
